# MineralOps — PT. Mineral Baturaja Indonesia

Sistem manajemen operasi tambang batubara berbasis AI.

## Fitur
- Input lapangan: Manajemen Air, Produksi OB, BBM/Solar
- Sinkronisasi ke Google Sheets
- Briefing AI harian (Gemini)
- Prediksi level air + cuaca 3 hari (Open-Meteo)
- Deteksi dini anomali operasional
- Chat AI tanya apa saja
- PWA — bisa diinstall di HP Android & iOS

## Versi
v1.3 · AI + Google Sheets + Cuaca

## Deploy
Hosted di Vercel. Untuk update: push ke branch `main`.
